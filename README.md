# memp

## Install
```sh
mkdir -p ~/.local/bin && curl -o ~/.local/bin/memp https://raw.githubusercontent.com/carter4299/memp/main/memp && chmod +x ~/.local/bin/memp
```

## Usage
```
cartersusi@linux ~> memp Alacritty
Total processes:        3
Total memory usage: .07 GB
cartersusi@linux ~> memp Firefox
Total processes:       11
Total memory usage: 1.43 GB
```
